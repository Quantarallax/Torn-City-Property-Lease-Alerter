# Torn City Property Lease Alerter
Allows you to set RAG values of days until lease expires, to give you sufficient time to send out lease extentions to tenants.

Install into TamperMonkey https://www.tampermonkey.net/

# Overview
On the properties page of Torn City MMORPG, it will highlight each of your properties based on user configurable values for days left before lease expires.
This allows you sufficient time to send out lease extentions.
